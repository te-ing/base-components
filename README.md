# Base Components

React 19, Storybook 10, TypeScript, pnpm 기반으로 만든 **공통 UI 컴포넌트 모노레포**입니다.

이 레포지토리는
**공통 UI 컴포넌트 구조 설계**, **Storybook 10 기반 문서화**, **pnpm + 모노레포 + (추후 Turborepo) 구조** 를 실험하고 발전시키기 위한 용도로 사용됩니다.

재사용 가능한 UI 컴포넌트를 패키지로 관리하고, Storybook을 통해 문서화/데모를 제공하며, 추후 이 컴포넌트들을 활용한 실제 서비스까지 확장하는 것을 목표로 합니다.
